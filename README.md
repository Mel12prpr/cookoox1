# CookBox

## ­ЪЊї рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃўрЃА рЃЉрЃљрЃќрЃљ
CookBox рЃљрЃарЃўрЃА рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЦрЃЋрЃћрЃДрЃюрЃўрЃА рЃЎрЃћрЃарЃФрЃћрЃЉрЃўрЃА рЃЉрЃљрЃќрЃљрЃА.

### ­ЪЊѓ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
- **ID**
- **рЃЎрЃћрЃарЃФрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ**
- **рЃЎрЃћрЃарЃФрЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃљ рЃџрЃљрЃЌрЃўрЃюрЃБрЃарЃў рЃљрЃАрЃЮрЃћрЃЉрЃўрЃЌ**
- **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљ**
- **рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљ**
- **рЃерЃћрЃЏрЃљрЃЊрЃњрЃћрЃюрЃћрЃџрЃў рЃўрЃюрЃњрЃарЃћрЃЊрЃўрЃћрЃюрЃбрЃћрЃЉрЃў**
- **рЃўрЃюрЃњрЃарЃћрЃЊрЃўрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃгрЃЮрЃЊрЃћрЃЉрЃљ рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ**
- **­ЪЊќ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ** (рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЎрЃџрЃћ рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ)
- **РЈ│ рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮ**

> **Рё╣№ИЈ рЃерЃћрЃюрЃўрЃерЃЋрЃюрЃљ:** рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃЮрЃљ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃЮрЃА. рЃЉрЃљрЃќрЃљрЃерЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃюрЃЏрЃћрЃЮрЃарЃЊрЃћрЃА рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃЎрЃћрЃарЃФрЃў, рЃЌрЃБ рЃЏрЃљрЃА рЃљрЃЦрЃЋрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃўрЃА рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ.

---

## ­ЪЈа рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў
рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА:
- ­ЪћЇ **рЃАрЃљрЃФрЃўрЃћрЃЉрЃЮ рЃЋрЃћрЃџрЃА**
- ­ЪЊѓ **рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃА**:
  - ­ЪЇ░ **рЃЊрЃћрЃАрЃћрЃарЃбрЃћрЃЉрЃў**
  - ­ЪЇ▓ **рЃЎрЃћрЃарЃФрЃћрЃЉрЃў**
  - ­ЪЦф **рЃгрЃљрЃАрЃљрЃ«рЃћрЃЏрЃАрЃћрЃЉрЃћрЃџрЃў**

### ­Ъћј рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃФрЃўрЃћрЃЉрЃљ
рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљрЃќрЃћ рЃЊрЃљрЃЎрЃџрЃўрЃЎрЃћрЃЉрЃўрЃАрЃљрЃА рЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃњрЃЋрЃћрЃарЃЊрЃў, рЃАрЃљрЃЊрЃљрЃф рЃЉрЃљрЃќрЃўрЃЊрЃљрЃю рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ рЃљрЃЏ рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў.

### ­Ъћј рЃАрЃљрЃФрЃўрЃћрЃЉрЃЮ рЃЋрЃћрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃЎрЃћрЃарЃФрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА _(рЃњрЃљрЃарЃЊрЃљ ID рЃЋрЃћрЃџрЃўрЃА)_.

### ­Ъј▓ рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃљ
рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃљрЃЊ (random) рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџ рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃћрЃю:
- ­ЪЊї **рЃЎрЃћрЃарЃФрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃА**
- ­ЪЊќ **рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљрЃА**
- РЈ│ **рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА**
- ­Ъќ╝ **рЃцрЃЮрЃбрЃЮрЃА**

---

## ­ЪїЇ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃќрЃљрЃарЃћрЃБрЃџрЃЮ
рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃЊрЃћрЃЉрЃљ рЃдрЃўрЃџрЃљрЃЎрЃў **"­Ъїј рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃЏрЃќрЃљрЃарЃћрЃБрЃџрЃЮ"**, рЃарЃЮрЃЏрЃћрЃџрЃќрЃћрЃф рЃЊрЃљрЃЎрЃџрЃўрЃЎрЃћрЃЉрЃўрЃАрЃљрЃА рЃњрЃљрЃўрЃ«рЃАрЃюрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў рЃњрЃЋрЃћрЃарЃЊрЃў, рЃАрЃљрЃЊрЃљрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃўрЃарЃЕрЃћрЃЋрЃА рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљрЃА.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
- ­ЪЄг­ЪЄф **рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮ**
- ­ЪЄ«­ЪЄ╣ **рЃўрЃбрЃљрЃџрЃўрЃљ**
- ­ЪЄФ­ЪЄи **рЃАрЃљрЃцрЃарЃљрЃюрЃњрЃћрЃЌрЃў**
- ­ЪЄ»­ЪЄх **рЃўрЃљрЃърЃЮрЃюрЃўрЃљ**

рЃДрЃЮрЃЋрЃћрЃџ рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљрЃА рЃћрЃЦрЃюрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃАрЃљрЃЏрЃќрЃљрЃарЃћрЃБрЃџрЃЮрЃА рЃњрЃљрЃЏрЃЮрЃарЃЕрЃћрЃБрЃџрЃў рЃЎрЃћрЃарЃФрЃўрЃА рЃцрЃЮрЃбрЃЮ, рЃарЃљрЃф рЃњрЃљрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃа рЃљрЃдрЃЦрЃЏрЃљрЃА. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃўрЃћрЃа рЃЦрЃЋрЃћрЃДрЃюрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЉрЃљрЃќрЃўрЃЊрЃљрЃю рЃЏрЃЮрЃўрЃФрЃўрЃћрЃЉрЃА рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА рЃљрЃЏ рЃЦрЃЋрЃћрЃДрЃюрЃўрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃЎрЃћрЃарЃФрЃћрЃЉрЃА.

---

## ­ЪЦд рЃўрЃюрЃњрЃарЃћрЃЊрЃўрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃФрЃўрЃћрЃЉрЃљ
рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃерЃћрЃћрЃФрЃџрЃћрЃЉрЃљ рЃљрЃўрЃарЃЕрЃўрЃЮрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃўрЃюрЃњрЃарЃћрЃЊрЃўрЃћрЃюрЃбрЃћрЃЉрЃў, рЃарЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃљрЃф рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЏрЃЮрЃФрЃћрЃЉрЃюрЃўрЃА рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃбрЃљрЃюрЃА рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃљрЃЏ рЃўрЃюрЃњрЃарЃћрЃЊрЃўрЃћрЃюрЃбрЃћрЃЉрЃА.
